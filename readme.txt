Allows a user to attach documents with Microsoft's Office Web Viewer or Google Document Viewer

== Description ==

Optimum Documents allows WordPress users to embed documents using either the Microsoft Office Web Apps Viewer or the Google Document viewer. Optimum Documents only requires a url parameter, but provides the option to specify height and width.

What makes Optimum different than other similar plugins is that the options given allow a user to embed quite a few more document types.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `optimum-docs.zip` through the Wordpress 'Plugins' menu
1. Activate the plugin.
1. Visit 'Settings->Optimum Documents' to learn more about how to use the plugin.
